Class: Message

Message

new Message()

validate時にエラーがあった時、エラーメッセージを表示するプラグイン

messageプラグインには以下の設定項目があります。

設定項目 型 説明 デフォルト値
appendMessage function メッセージ要素配置関数。メッセージ要素の配置を行う関数を指定します。第1引数にメッセージ要素(DOM)、第2引数にメッセージ追加対象要素、第3引数にプロパティ名が渡されます。指定しない場合は、メッセージ追加対象要素の後ろに追加します。 なし
wrapper string メッセージを出力する要素のタグ名またはタグ生成文字列。'li'や、'<span class="error-msg">'のような指定ができ、指定された文字列から生成した要素が各メッセージ要素になります。 なし(テキストノードとして表示)

プラグインの設定方法は、h5.ui.FormController.setSettingをご覧ください。
